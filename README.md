> 无聊时候，读读源码。

lodash version: 4.17.4

环境中需要已经安装有 [`lodash-cli`](https://github.com/lodash-archive/lodash-cli)。

首先安装依赖包：

```bash
npm i
```

执行:

```bash
npm run getlodash
```

可以拉取模块化的 `lodash` 代码到目录 `./lodash` 下。

执行：

```bash
gulp copy
```

将代码分门别类。

如果想要重新开始，可以拉取 `lodash` 代码并进行分类，执行：

```bash
npm run generate
```
